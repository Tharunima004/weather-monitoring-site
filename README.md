# Weather Monitoring Site
 
A responsive weather monitoring web app built with **HTML, CSS, and JavaScript** that fetches real-time weather data for any city and displays current conditions in a clean, easy-to-read interface.

## ✨ Features
- Search weather by city name
- Displays temperature, humidity, wind speed, and weather condition
- Weather icons/visuals based on current conditions
- Responsive design for mobile and desktop
- Error handling for invalid city names

## 🚀 Getting Started
 
### Run Locally
1. Clone the repository
```bash
   git clone https://github.com/Tharunima004/weather-monitoring-site.git
```
2. Open `index.html` in your browser.
3. *(If using an API key)* Add your API key in `script.js` before running.
## 📖 How It Works
- The user enters a city name and submits the search.
- JavaScript sends a request to the weather API using `fetch()`.
- The response (temperature, humidity, conditions, etc.) is parsed and displayed dynamically on the page.
- Errors (e.g., invalid city) are caught and shown to the user.
